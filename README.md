## PolarLobby Plugin

Een Paper plugin voor een veilige lobby-ervaring. 

### Functies
- **Spawn systeem** - Automatische teleportatie bij join en respawn
- **Bescherming** - Geen PvP, honger, of blok interactie voor spelers
- **Void teleportatie** - Automatische terugkeer naar spawn bij vallen
- **Bouw permissies** - Admins kunnen spelers tijdelijk bouw rechten geven

### Commando's
- `/pl setspawn` - Stel spawn locatie in
- `/pl allowbreak [speler]` - Toggle bouw permissie
- `/pl voidtp` - Configureer void teleportatie hoogte
- `/pl reload` - Herlaad configuratie

### Vereisten
- Paper 1.21.11
- ProtocolLib
- Java 21
